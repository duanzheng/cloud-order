## Clone it

```bash
$ cd path/to/root
$ git clone https://github.com/zce/weapp-boilerplate.git [project-name] --depth 1
$ cd [project-name]
```

## Install dependencies

```bash
$ npm install
```

## Usage

```bash
# build for production with minification
$ npm run build

# watch file changes
$ npm run watch
```

then, open `dist` in wechat devtools
